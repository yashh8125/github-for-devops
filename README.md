HashForge – GUI Hash Generator in Python

A hash function converts one value into another fixed-length value. Hashing is widely used in computer science for cryptography, data integrity verification, checksums, compression, and indexing. In cybersecurity, hashing helps protect sensitive information by transforming original data into unreadable hash values.

This project is a GUI-based hash generator developed in Python. It allows users to generate hashes using multiple cryptographic algorithms and can also help in understanding rainbow table generation concepts.

Features:
Generate hashes for any input string
Supports multiple hashing algorithms:
MD5
SHA1
SHA2
SHA3
Blake2a
Blake2b
Simple and user-friendly GUI
Useful for learning hashing and rainbow table concepts

Technologies Used:
Python
Tkinter
hashlib


How to Run the Script:
cd Hash-Generator-in-Python
python hash_generator.py
